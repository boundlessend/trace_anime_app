# TraceAnime - PRD

Документ продукта: что реализовано сейчас и что рассматривается дальше.
Раздел 5 (идеи) - черновик, не утверждён.

## 1. Обзор

TraceAnime - утилита в menu bar macOS для поиска аниме-сцены по кадру через API [trace.moe](https://trace.moe/).
Пользователь даёт картинку (URL, буфер обмена, drag and drop, файл), приложение возвращает аниме, эпизод, таймкод и превью момента.

- Платформа: macOS 14+, только menu bar (`LSUIElement`).
- Аудитория: те, кто хочет быстро опознать аниме по скриншоту или кадру.
- Распространение: DMG из GitHub Releases, ad-hoc подпись.

## 2. Архитектура и стек

- Swift 5.9 tools, SwiftUI, SwiftPM, единый executable target `TraceAnime`.
- Функциональный стиль: модели - `struct`/`enum`, бизнес-логика - чистые функции, классы только для внешних систем (API-клиент, провайдеры ввода, хранилища).
- Хранение: `UserDefaults` (`AppSettingsStorage`, `LibraryStorage`).
- Сеть: `TraceMoeClient` поверх `URLSession`, таймаут 30s, кэш отключён.
- Локализация: функция `t(...)` (`Localization.swift`), языки EN / RU / FR.
- CI/CD: build (debug+release), swift-format lint, release по тегу `v*` (сборка DMG), Dependabot.

## 3. Реализовано (текущая версия)

### Ввод и поиск
- Поиск по URL изображения.
- Поиск изображения из буфера обмена.
- Drag and drop: файл или изображение.
- Выбор файла с диска.
- Опция `cutBorders` (обрезка чёрных полос перед поиском).
- Фильтр по `anilistID`.
- Лимит загрузки 25 MB, превью-картинка до 10 MB.

### Результаты
- Список совпадений: название (romaji / native / english), эпизод, таймкод `from-to`, similarity %.
- Inline-воспроизведение превью-видео момента, mute.
- Размер превью S / M / L.
- Открытие страницы аниме на AniList с копированием ссылки.

### Библиотека
- История последних 20 поисков с исходным изображением.
- Избранное (favorites) с исходным изображением, toggle.

### Аккаунт и обслуживание
- Опциональный API-token trace.moe (`x-trace-key`), хранится в настройках.
- Просмотр квоты: id, quota used/total, concurrency, priority.
- Очистка кэша превью.

### Платформа
- Многоязычный UI EN / RU / FR, переключение на лету.
- Liquid-glass оформление, перетаскиваемый sticky-копирайт.

## 4. Ограничения и пробелы

- Нет нотаризации / Developer ID: при первом запуске срабатывает Gatekeeper.
- Историю нельзя чистить или удалять записи вручную (только авто-trim до 20).
- `SearchQueue` (actor) сейчас просто проксирует клиент - реальной очереди/ограничения конкурентности нет.
- Открытие результата есть только для AniList; нет MyAnimeList и копирования текстовых данных.
- Нет автоматических тестов (в т.ч. для капризного JSON trace.moe: int-or-string, anilist id-or-info, episode number/text/list).
- Нет автообновления приложения.

Порядок реализации: сначала дешёвые P0, затем флагман и инфраструктура.

## 6. Отложено (бэклог)

- Blur для NSFW-превью (`isAdult` уже в данных).
- Нотаризация релиза (Developer ID) - требует платный Apple Developer аккаунт.
- Уведомление о завершении долгого поиска при закрытом popover.
